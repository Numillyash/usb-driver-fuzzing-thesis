# Experiments Layer

Данный каталог содержит воспроизводимый слой практических экспериментов ВКР по устойчивости USB-стека ОС к безопасным аномалиям уровня descriptor/enumeration.

## Структура

- `cases/` — формализованные JSON-кейсы экспериментов.
- `templates/` — шаблоны фиксации результатов.
- `results/` — каталог для накопления полученных результатов и артефактов.

## Принципы

1. Источник сценария — только JSON-кейс в `cases/`.
2. Выполнение фокусируется на RP2040 как управляемом USB-устройстве.
3. Целевые платформы анализа — Windows и Linux.
4. Сценарии ограничены безопасными аномалиями дескрипторов и процесса перечисления.
